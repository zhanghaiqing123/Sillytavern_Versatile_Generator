一、概述

多功能生成器是一款运行于 SillyTavern 酒馆助手框架下的增强型脚本工具。
它提供了独立于主聊天的生成通道、可视化的生成器工坊、自动化的 XML 排版与世界书注入能力，旨在帮助用户在不污染主对话上下文的前提下，高效生成角色、物品、建筑、事件等结构化内容，并将其无缝写入世界书。

---

二、核心功能

1. 独立 API 通道

· 插件拥有独立于主聊天的 API 配置面板。
· 支持 OpenAI 兼容协议与 Claude 协议。
· 用户可为主聊天使用高性能模型（如 GPT-4o、Claude），同时在插件内配置低成本模型（如 Gemini Flash）进行生成任务。
· 主对话不受任何影响：不占用主对话 Token，不写入聊天记录。

2. 上下文智能抓取

· 支持手动控制抓取最近 N 条聊天记录作为生成参考。
· 内置剧情提取规则：
  · 可指定 AI 正文与用户输入的提取标签（如 <content>）。
  · 自动剥离大模型的 <think> 思维链内容。
· 仅将纯净的文本内容送入生成器，防止 AI 发散偏题。

3. 生成器工坊（Builder Mode）

用户可自由创建、编辑、删除生成器，支持以下操作：

· 可视化字段管理：自由增删生成字段，支持长按拖拽排序。
· 系统提示词编辑：为每个生成器独立配置 System Prompt。
· 导入 / 导出：生成器以 JSON 格式导出，可一键导入分享。
· 默认生成器恢复：对内置生成器的修改可一键恢复出厂设置。

4. 全自动 XML 排版与解析

· 用户只需定义字段名称与提示词。
· 脚本在底层自动组装严格的 XML 格式约束发送给 AI。
· AI 返回后，自动按字段切割并回填至表单。
· 无需手写 XML 标签，全程可视化操作。

5. 生成结果审核与写入系统

· 可视化编辑：所有生成字段均显示在输入框中，支持手动修改。
· 重新生成：支持基于上次修改后的内容重新调用 AI 生成。
· 追加说明：提供无格式限制的补充文本框。
· 一键写入世界书：
  · 自动分离关键词与正文。
  · 支持 🟢绿灯（关键词触发） / 🔵蓝灯（常驻）两种触发方式。
  · 支持选择注入位置（角色定义前 / 后）与顺序权重。
  · 自动查重与覆盖：检测同名条目并提示覆盖。

五、安装与配置

环境要求

· SillyTavern 已正确安装并运行
· 已安装 TavernHelper（酒馆助手） 扩展

安装步骤

1. 打开 SillyTavern → 酒馆助手 → 脚本管理器 → 新建全局脚本。
2. 选择下载的本脚本文件
3. 保存脚本并刷新页面。
4. 点击右下角悬浮球或顶部扩展按钮中的「生成器」打开主面板。
5. 进入「设置与配置」填写 API 信息（地址、密钥、模型等）。

API 配置说明

参数 说明
API 地址 支持 OpenAI 兼容地址，如 Gemini / 本地推理服务
API 密钥 对应服务的密钥
模型名称 如 gemini-2.5-flash-lite、gpt-3.5-turbo
温度 / Top P / 惩罚系数 控制生成随机性与多样性
最大 Token 数 限制单次生成长度（建议 12000 以上，低了会被直接截断）
